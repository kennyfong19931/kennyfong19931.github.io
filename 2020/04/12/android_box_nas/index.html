Android 盒自製 NAS

發表於 2020-04-12 更新於 2024-10-19 分類於 Android 盒 本文字數： 2.2k 閱讀時長 ≈ 4 分鍾

呢幾年有唔少公司出過 Android 盒，有 D 已經無任何 support，好似我手上呢隻香港寬頻送嘅樂視盒子，樂視直頭清咗盤，要自己另外裝 APP 先睇到片。

由於樂視個 rom 本身係俾用家自己裝 APP，又有個 USB Type-A 頭，想改裝做 NAS 可以話係無難度。

今次改裝嘅核心係一個叫 Servers Ultimate 嘅 APP，任何 Android 裝置都可以用佢變成 server。

由於今次係將個 Android 盒廢物利用，所以未必可以好似而家市面上買得到嘅 NAS 有咁多功能。例如 RAID，虛擬機呢類功能就未必做到啦。

Servers Ultimate 介紹

XDA 原帖
Google Play (免費版) Google Play (付費版)

Servers Ultimate 支持 Android 4.1 以上嘅裝置，包括 70 款以上服務可以用。只要裝好主程式 + Service Pack 就可以開到想用嘅服務。免費版同付費版嘅唯一分別係免費版只可以同時開3個Service，如何要同時用超過3個 Service 嘅就要用付費版。主程式入面無包哂所有 Service，有 D Service要裝埋 Service Pack 先用到 (免費)。詳情可以睇下面條 list。

使用方法

打開 APP 會見到呢版，可以直接喺 Server list 揀你想要嘅 server 之後 Start 就得

Start 過嘅 server 可以喺 Servers 睇到

Click 落去可以有更詳細嘅設定，每款 Servers 都有唔同嘅設定，我就唔逐個 list 出嚟啦

NAS Service

一般家用 NAS 最常用嘅功能都係喺唔同嘅裝置 (電腦 / 電話 / 平板) 上都睇到自己嘅文件 / 音樂 / 影片，只要用 Android 盒開番對應嘅 Service 就可以，將 Server 嘅 root path set 做以下嘅位就得。

內置儲存位置 /storage/emulated/legacy
外置儲存位置 (USB) /storage/external_storage/sda1

  • 檔案傳輸 - iSCSI/NFS/WebDAV/SMB/FTP/SFTP
  • 影音 - NFS/WebDAV/DLNA/UPnP
